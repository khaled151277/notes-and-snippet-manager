1. Introduction

Welcome to the Notes & Snippets Manager! This application helps you organize your thoughts, ideas, and useful code snippets in one place. It features a rich text editor for notes and a code editor with syntax highlighting for snippets.

2. Interface Overview

The main window is divided into two main sections:

  • Sidebar (Left): Contains the search bar, tabs for "Notes" and "Snippets" lists, and buttons to create new items.
  • Content Area (Right): This is where editor tabs for individual notes and snippets are opened.

The status bar at the bottom may display error messages related to database operations.

3. Managing Notes

Use the "Notes" tab in the sidebar to manage your rich-text notes.

3.1 Creating/Editing Notes

  • Create:** Click the "New Note" button (Ctrl+N) to open a new, empty note editor tab.
  • Open:** Click or double-click a note in the list to open it in an editor tab.
  • Title:** Enter a title for your note in the top input field.
  • Content:** Write your note content in the main editor area below the formatting toolbar.

3.2 Formatting Notes

The toolbar above the note content area provides standard formatting options:

  • Basic Styles:** Bold (Ctrl+B), Italic (Ctrl+I), Underline (Ctrl+U). These are toggle buttons.
  • Alignment:** Align Left (Ctrl+L), Align Center (Ctrl+E), Align Right (Ctrl+R). Only one can be active at a time for the current paragraph.
  • Font:** Select a font family and size using the dropdown and spin box.
  • Color:** Click the color button to choose a text color.

Formatting applies to selected text or the text you type after changing the format.

3.3 Working with Links

  • Creating/Editing Links:**
    • Select the text you want to turn into a link (or place the cursor where you want to insert a new link).
    • Click the "Insert/Edit Link" button (icon looks like chain links, Ctrl+K) on the toolbar.
    • A dialog will appear. Enter the "Display Text" and the "Link URL" (e.g., https://www.example.com or mailto:user@example.com).
    • If you selected an existing link created this way, the dialog will be pre-filled, allowing you to edit or remove the link.
  • Clicking Links:** Click on a link (text that is typically blue and underlined) to open it in your default web browser or email client.
  • Note:** Links must be created using the "Insert/Edit Link" button to be reliably clickable. Plain text URLs or links from older versions might not be interactive.

4. Managing Snippets

Use the "Snippets" tab in the sidebar to manage your code snippets.

4.1 Creating/Editing Snippets

  • Create:** Click the "New Snippet" button (Ctrl+Shift+N) to open a new, empty snippet editor tab.
  • Open:** Click or double-click a snippet in the list to open it in an editor tab.
  • Title:** Enter a title for your snippet in the top input field.
  • Code:** Enter or paste your code into the main editor area.

4.2 Syntax Highlighting

  • Select the appropriate programming language from the dropdown menu above the code editor.
  • Syntax highlighting will be applied automatically based on the selected language. Supported languages include Python, JavaScript, HTML, CSS, SQL, Java, C++, C#, PHP, Ruby, Go, and plain Text.

4.3 Copying Code

  • Click the "Copy Code" button (Ctrl+Shift+C) located in the button row below the editor to copy the entire code snippet to your system clipboard.

5. Common Features

5.1 Titles and Tags

  • Both notes and snippets require a title or content/code to be saved.
  • You can add comma-separated tags in the "Tags" input field below the main content area to categorize your items (e.g., python, database, example). Tags are included in searches.

5.2 Saving and Deleting

  • Saving:** Click the "Save" button (Ctrl+S) to save the current note or snippet. New items will be added to the list, and existing items will be updated. The asterisk (*) in the tab title will disappear upon successful save.
  • Deleting:** Click the "Delete" button (Ctrl+Delete) to permanently delete the currently open note or snippet. You will be asked for confirmation. The "Delete" button is only visible for saved items.

5.3 Searching

  • Type your search query into the search bar at the top of the sidebar.
  • The application will search in real-time through the titles, content/code, and tags of both notes and snippets.
  • The lists will update automatically to show matching results.
  • Clear the search bar to show all items again.

5.4 Working with Tabs

  • Each opened note or snippet appears in its own tab in the content area.
  • Tabs with unsaved changes will have an asterisk (*) next to their title.
  • Close a tab by clicking the 'x' button on the tab or by pressing Ctrl+W.
  • If a tab has unsaved changes, you will be prompted to Save, Discard, or Cancel before closing.
  • When closing the main application window, you will be prompted similarly for any open tabs with unsaved changes.

6. Keyboard Shortcuts

  • Ctrl+N: New Note
  • Ctrl+Shift+N: New Snippet
  • Ctrl+S: Save current note/snippet
  • Ctrl+Delete: Delete current note/snippet
  • Ctrl+W: Close current tab
  • Ctrl+B: Toggle Bold (Note Editor)
  • Ctrl+I: Toggle Italic (Note Editor)
  • Ctrl+U: Toggle Underline (Note Editor)
  • Ctrl+L: Align Left (Note Editor)
  • Ctrl+E: Align Center (Note Editor)
  • Ctrl+R: Align Right (Note Editor)
  • Ctrl+K: Insert/Edit Link (Note Editor)
  • Ctrl+Shift+C: Copy Code (Snippet Editor)
